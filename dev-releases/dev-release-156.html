Exciting Updates Coming Soon for Our Blockchain Platform!

Hey Community!

We’re thrilled to share some insights on what’s ahead in the next phase of our development journey. Our team has been in continuous, productive discussions with our stakeholders, carefully planning a series of groundbreaking advancements that will drive our platform forward. 

Here’s what’s on the horizon:

Fee Customization on BlockDAG Blockchain

One of the primary upgrades we’re aiming to introduce is fee customization within our BlockDAG blockchain. This feature will bring flexibility to transactions, allowing users to tailor fees based on network conditions and transaction priorities. By introducing a scalable fee structure, we aim to optimize the network’s resource allocation, reduce congestion during peak periods, and create a fair system that empowers users with the flexibility they need.
This advancement will support a wide range of transaction types, from micro-payments to high-value exchanges, making the BlockDAG network adaptable for everyone—from individual users to large enterprise clients.

Upcoming Modules: Contract Listing, Token Listing, Nodes Module & ERC-20 Contract Verification

We’re finalizing the scope for our next release, and we’re excited to announce that the following key modules are set to be included. 

Let’s break down each feature and how it will enhance the overall user experience on our platform:

  • Contract Listing Module The contract listing module will be a one-stop repository for all active smart contracts deployed on our network. Users will have access to detailed information, such as:
    • Contract Addresses: Unique identifiers for each contract, simplifying contract access and interactions.
    • Contract Names and Types: Easily identify specific contracts, their purpose, and their functionality.
    • Creation Timestamp and Metadata: Know when each contract was deployed and access supplementary details provided by developers.
       
  • By offering powerful search and filtering options, the contract listing module will make it simple for users to locate and interact with contracts based on various criteria—such as type, popularity, or date of deployment. This is designed to promote transparency and improve the overall accessibility of smart contracts on our network, benefiting developers and users alike.
     
  • Token Listing Module Our dedicated token listing module will act as a comprehensive index of all tokens created and deployed on our blockchain. Token listings will include essential details such as:
    • Token Name and Symbol: Easy identification of each token.
    • Total Supply and Circulating Supply: Track how much of a token is in circulation, aiding in market analysis and user trust.
    • Market Cap and Price History: For users and investors interested in token performance and value.
       
  • Additionally, the token module will enable users to browse recent token transactions, view token distribution statistics, and track changes over time. This feature is aimed at creating transparency in token creation and activity, helping token holders make informed decisions, while also giving developers a platform to showcase their tokens and increase their visibility.
     
  • Nodes Module The nodes module will provide a detailed, interactive map of the network’s node ecosystem. By offering a live view of each node’s status, users will be able to access:
    • Node Location: A visual representation of node distribution across different geographic regions.
    • Performance Metrics: Key metrics like response time, transaction throughput, and latency, enabling insights into network health.
    • Connection Status and Uptime History: Monitor which nodes are active, which are experiencing downtime, and view historical uptime.
       
  • This module will also offer real-time monitoring of node-to-node connections and network topology, allowing users to observe how network data flows and identify any issues with connectivity. By promoting transparency into the decentralized infrastructure, the nodes module will give stakeholders a clear understanding of the network's resilience and reliability.
     
  • ERC-20 Contract Verification In an effort to boost security and trust within the ecosystem, we’re introducing an ERC-20 contract verification module. This tool will allow developers to submit their source code for verification, enabling users to identify contracts that have been audited and approved. The verification process will cover:
    • Source Code Authentication: Ensures that the source code corresponds to the deployed bytecode.
    • Security and Compliance Checks: Evaluate contracts for common vulnerabilities and standards compliance.
    • Trust Badge for Verified Contracts: Displayed next to verified contracts, offering users assurance and encouraging safe interactions with tokens.
       
  • Verified contracts will be marked with a “Trust Badge,” signaling to users that the contract has undergone a thorough review and aligns with platform standards. This module aims to foster security and transparency within our blockchain by reducing the risk of malicious or poorly written contracts.

Automation of the EVM Explorer

Our team is currently hard at work to fully automate our EVM (Ethereum Virtual Machine) explorer. This automation will streamline blockchain data tracking, enabling faster, more accurate data updates, and reducing manual intervention. By creating a seamless, automated data pipeline, we’re ensuring that our explorer becomes a reliable, up-to-the-minute source of information.
This enhancement will allow developers to access real-time data, explore detailed smart contract analytics, and get a clearer view of network behavior and trends, making it an essential tool for anyone involved in the ecosystem.

Automated Report Generation

Alongside the EVM explorer automation, we’re also developing automated report generation. This feature will allow us to produce comprehensive reports efficiently, offering detailed analytics and insights without the need for extensive manual compilation. These reports will be valuable for stakeholders, developers, and users who want a quick overview of network health, performance, and activity.
Automated reporting will include metrics on transaction volumes, node health, smart contract usage, and more. This enhancement will not only save time but also ensure that the information stakeholders receive is consistent, accurate, and data-driven, providing a solid basis for making strategic decisions.

What's Next?

Our discussions with stakeholders are ongoing, and we’re fine-tuning these plans to make sure they meet the high standards of our platform and community. Each of these enhancements aligns with our commitment to making our platform more powerful, adaptable, and user-centric.
As we move forward, we’ll be sharing updates, sneak peeks, and timelines to keep our community in the loop. We’re committed to delivering quality, efficiency, and a better user experience with every new feature and upgrade.

Stay connected—there are incredible developments on the way!
