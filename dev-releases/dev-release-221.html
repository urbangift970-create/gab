Awakening Network Launch & Major BlockDAG Upgrades

We’ve successfully launched the Awakening network on 25th September 2025.
 

This release delivers one of the largest milestone upgrades in BlockDAG history. Awakening introduces protocol-level transformations (account model change, account abstraction, runtime upgradability), optimised user-facing dApps (Lottery & BDAG Reflection), an advanced developer IDE, and a significantly enhanced Explorer with NFT support, real-time analytics, and WalletConnect integration.

These updates reinforce BlockDAG’s vision of being a developer-first, user-centric ecosystem with stronger security, flexible functionality, transparent data, and enterprise-grade readiness.

Purpose

  • Ecosystem Expansion: Introduce features that broaden BlockDAG’s appeal to both developers and everyday users (NFTs, Lottery, Reflection).
  • Protocol Strengthening: Upgrade the core to a more scalable, flexible, and EVM-aligned foundation.
  • Developer Enablement: Provide modern tooling with the BlockDAG IDE, empowering smart contract innovation.
  • Transparency & Analytics: Deliver real-time, drill-down dashboards and enhanced Explorer modules.

Key Highlights

Explorer Enhancements

NFT Module: Latest transfers, mint lists, NFT detail pages with metadata, images, videos & GIF support.

Advanced Filters: Blocks, transactions, token transfers with CSV export for auditors/analysts.

Graphs & Stats: Live dashboard with drill-down time-series charts (TPS, gas, addresses, contracts, fees).

WalletConnect Integration: Connect wallets directly for secure, seamless transactions.

UX Upgrades: Copy-to-clipboard, responsive layouts, deep linking, improved tabular views.
 

Blockchain Protocol Core

UTXO → Account-Based Model: Simplified smart contract flows, aligned with EVM standards.

Account Abstraction (EIP-4337): Smart wallets as primary accounts, gas sponsorship, batching, programmable recovery.

Runtime Upgradability: Modular upgrades without disruptive hard forks, enabling faster security patches and feature rollouts.
 

dApps

BDAG Reflection DApp: Trustless, on-chain reflections; users send BDAG and receive an equal amount instantly.

Lottery DApp: BDAG-based bidding, auditable winner selection (VRF recommended), automatic reward distribution, and Explorer visibility.
 

BlockDAG IDE

Full-featured, open-source IDE with syntax highlighting, debugging, testing, deployment wizard, plugin support (Hardhat/Truffle/Remix), and wallet interoperability.

Step-through debugging, coverage reports, gas estimation, dry-run deployments.

Extensible plugin architecture and developer-first documentation.

Performance Validation

  • Testing Scope: Explorer load tests (charts, block/transaction queries, API endpoints), Reflection & Lottery contract execution, IDE plugin integration.
  • Metrics Captured: Latency (p50/p95/p99), throughput, error thresholds, caching efficiency.
  • Outcome: Systems sustained peak simulated traffic with zero critical failures, validating resilience for Awakening-scale activity.

Feedback & Collaboration

Team is now coordinating with internal stakeholders and community to make the release more successful and stable. 

Users: Share Explorer/NFT usability insights.

Operators: Validate the user guides and documentation.

Developers: Test & contribute to this network much better.

Final Thoughts

Awakening marks a turning point for BlockDAG. By unifying protocol upgrades, transparent data visibility, user-first dApps, and developer tooling, this release sets the stage for real-world adoption.

It’s not just an upgrade—it’s the foundation of a new era. Awakening empowers developers, users, and operators alike to engage with a network that’s faster, more secure, and truly community-driven.

BlockDAG is no longer in build mode—it’s in execution mode. Awakening is the beginning of a stronger, more resilient future for decentralized ecosystems.
