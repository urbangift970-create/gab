Dev Release 121

Release Date: September 18, 2024

  • Conclusion

Greetings BlockDAG Community,

BlockDAG Explorer Development Release Notes
We are excited to announce the latest updates to the BlockDAG Explorer, now featuring enhanced UTXO and EVM Transactions support for both transaction and address pages. This update brings several improvements and new functionalities to the platform, making it easier for users to monitor, manage, and interact with the BlockDAG network. The following sections outline the key aspects of this release:
1. New Block Page for UTXO and EVM Transactions
The BlockDAG Explorer now provides a detailed Block Page for both the UTXO (Unspent Transaction Output) and EVM (Ethereum Virtual Machine) transactions. This page gives users comprehensive insights into individual blocks, making it easier to explore both transaction models and understand the rewards and transactions associated with each block.
Key Features:

  • Block Hash and Number: Each block is identified by its unique Block Hash and Block Number, ensuring traceability within the BlockDAG network. Users can search by block number or hash to quickly find specific blocks.
  • Block Rewards: Information on the rewards associated with mining or validating the block is clearly displayed. This includes the BDAG rewards for block producers, giving insight into the incentive structure of the network.
  • Block Details: The block page includes essential details such as the timestamp, the number of transactions included in the block, and any fees collected from those transactions.
  • Transaction List: Each block also displays a list of transactions it contains, complete with Txn Hash, From and To addresses, and the Value transferred in each transaction. This provides transparency into the activity that occurred within the block.


2. Address Pages with Comprehensive Balance Information
Users can now explore address pages, which offer a detailed breakdown of the balance and transaction history associated with each account. This feature supports both UTXO and EVM Transactions, providing a unified experience across transaction types.
Key Features:

  • Address Details: The address page presents the full address along with its Balance in BDAG. Users can view their balances with precision and understand their total holdings.
  • Recent Transactions: A list of recent transactions is displayed, allowing users to view their transaction history. Each entry includes the Txn Hash, Block Number, and transaction Value for full transparency.
  • BlockDAG Value: The address page also provides real-time data on the current value of the BlockDAG token (BDAG), allowing users to easily track the market value of their holdings.
  • First and Last Transaction: Information about the first and last transactions associated with the address is also included for historical tracking.


3. Enhanced User Interface (UI) and Usability
The updated BlockDAG Explorer interface has been designed to improve user experience and navigation:

  • Navigation Menu: The left-hand side menu now provides quick access to essential features such as Home, Blocks, Transactions, and a Faucet for acquiring test tokens. The Add BDAG Network option is also available for easy integration into MetaMask.
  • BlockDAG Price Indicator: The real-time BDAG price is prominently displayed at the bottom of the menu, keeping users updated on market conditions.
  • Clear, Modern Layout: The explorer has adopted a modern, sleek dark theme with neon accents, improving readability and overall aesthetic appeal.


4. Key Use Cases

  • Transaction Monitoring: Users can easily track their transaction status, ensuring that transfers, swaps, or other interactions with the BlockDAG network are proceeding as expected. This includes EVM transactions, where smart contracts and token transfers are key elements.
  • Balance Overview: Wallet holders can monitor their balances, including historical transaction data, to keep tabs on spending, receiving, and managing funds.
  • Network Insights: Developers and network participants can gain insights into the latest blocks, the status of transactions, and the health of the network in a clean, organized manner.

Conclusion

This release of the BlockDAG Explorer emphasizes user experience improvements, better visualization of transaction models, and a modernized, user-friendly interface. Whether users are interacting with the UTXO model or EVM transactions, they can expect clear, concise, and comprehensive data presentation that allows for efficient network navigation.
Stay tuned for future updates as we continue to evolve the BlockDAG Explorer with more features and performance enhancements!
